### 即时备份

即时备份（on-demand）也就是单次的备份任务，对应的资源对象为 `backup`。

#### 创建

```shell
$ velero backup create
```

可选的 flags 包括：

**--default-volumes-to-restic**

默认为 true

**--exclude-namespaces**

**--exclude-resources**

**--from-schedule**

**--include-cluster-resources**

**--include-namespaces**

**--include-resources**

**--label-columns**

**--labels**

**--ordered-resources**

**--output**

**--selector**

**--show-labels**

**--snapshot-volumes**

**--storage-location**

**--ttl**

**--volume-snapshot-locations**

**--wait**
