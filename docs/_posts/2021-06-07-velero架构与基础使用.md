## 备份

### 即时备份

即时备份（on-demand）也就是单次的备份任务，对应的资源对象为 `backup`。

#### 创建

```shell
$ velero backup create
```

可选的 flags 包括

- --default-volumes-to-restic   optionalBool

  默认为 true，即由 restic 备份所有的 Pod 卷，优先级小于 velero install 中的对应参数

- --exclude-namespaces   stringArray

  指定排除的 namespace

- --exclude-resources   stringArray

- --from-schedule   string

- --include-cluster-resources   optionalBool

- --include-namespaces   stringArray

- --include-resources   stringArray

- --label-columns   stringArray

- --labels   mapStringString

- --ordered-resources   string

- --output   string

- --selector   labelSelector

- --show-labels   

- --snapshot-volumes   optionalBool

- --storage-location   string

- --ttl   duration

- --volume-snapshot-locations   strings

- --wait
